<h1>nooble - Simple Google clone project</h1>

Functionality:
 - Full-text indexing of a target page
 - Search for required word/fraze    

Frameworks 
 - Spring Boot
 - Spring MVC
 - Spring IoC
 - Lucene
 - Gradle
 - FreeMarker
 - Jsoup
 - lsf4j
 - java concurrency
 - java 8 (lambda, stream)
 
Web-server
 - Apache Tomcat 

IDE
 - InteliJ Idea 15
